This pertains to the DBHandling.java file and corresponding PHP files.
To run this library, you will need to relocate the PHP files to your localhost web server folder. If you are using IIS, this will be in inetpub/wwwroot, if you are using WampServer, it will be in your wamp/www folder. Other servers, you will need to consult your vendor documentation. 

API Reference:
To read documentation on how to use the DB Handling Library, please go to the API folder and open Index.html as this has the compiled JavaDoc.
